在線版 還是 離線版？

兩個版本共用相同的家族樹功能。差別在於你如何儲存資料，以及是否與家人共享。

🌐 在線版

與家人一起保存記憶

透過 Google Drive 雲端儲存，讓家人可以隨時在任何裝置上查看或協作。

  • ☁️儲存至自己的 Google Drive，任何裝置皆可開啟
  • 👥邀請家人成為協作編輯者（Editor）或只讀查看者（Viewer）
  • 🔄在手機、平板、電腦之間同步共享同一棵家族樹
  • 🔑需要 Google 帳號登入，資料存放於你自己的 Drive
  • 🖨️匯出為圖片或 PDF
開啟在線版
🔒 離線版

更私密，存於你的電腦

無需帳號，無需網絡。資料可由你自行另存為 JSON 檔案，保存到你選擇的電腦位置。

  • 🔒資料主要保留在你的電腦，不經平台集中儲存
  • 💾可另存為 JSON 檔案，自行選擇電腦上的保存位置
  • 📵無需 Google 帳號，亦不依賴網絡連線
  • 🖥️以單機使用為主，不支援自動多裝置同步
  • 🖨️同樣支援匯出為圖片或 PDF
⬇ 下載離線版 (.html)

下載後，請以 Chrome / Edge 開啟此 HTML 檔案使用；資料可另外儲存為 JSON 檔案。

兩個版本的家族樹核心功能相同，主要差別在儲存方式與共享能力。
